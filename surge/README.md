## 安装

1. install node
2. install yarn
3. cd workflow dir, and yarn install
4. Surge开启httpAPI, alfred workflow配置httpAPI


## 注意
Surge Mac 4.0.0、Surge iOS 4.4.0开始提供HTTP API, 也就是该版本之前的均不支持
